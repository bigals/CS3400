The ID Selector

The ID Selector allows specific elements to be selected on the page for styling. IDs are generally unique of the page, therefore the ID selector is best used when single elements need to be styled. The Class Selector

The class selector is used to apply styling to all elements that have a particular class assigned to them. Since classes generally are applied to a groupd of elements, the class selector is best used for styling multiple objects the exact same way.

The Element Selector

The Element selector allows the CSS styling to be applied to paritcualr HTML elements on the page. The selector ignores any classes or IDs that are associated with elements, and simply applies them to all of the specified elements.

Mixed Selector and Miscellaneous Selectors

All the selectors mentioned above can be used together to select subsets, or specific elements with any styling that is desired. Furthermore, other selectors can be used to style events such as hover or active.

Browser Compatability Testing

Browserling

Browserling requires a URL to a website to be applied, and once entered, you can test the full interface of your site on multiple browsers for a set period of time. Once the time has expired, jusr re-enter the URL of your site, and you can continue testing compability.

Supported Browsers:
  • IE9
  • Chrome(1.0-37.0)
  • Firefox(3.0-32.0)
  • Safari(4.0-5.1)
  • Opera(10.0-24.0)

Spoon

Spoon runs VMs in your browser in order to allow you to test your web site on most major browsers, including the Opera Mobile and Firefox mobile browsers.

Supported Browsers:
  • IE(6-11)
  • Chrome(11-37)
  • Firefox(2-32)
  • Safari(3-5)
  • Opera(9-20)
  • Firefox Mobile(2, 5)
  • Opera Mobile(6, 11.5, 12)

BrowserStack

BrowserStack runs VMs in browser to test almost all browsers, including various mobile device resolutions as well. BrowserStack is a paid monthly subscription service, but does have a Free Trial process, which doesnt involve credit card information of any kind. BrowserStack could be usefull in testing certain browsers that are not supported by Browserling or Spoon.

Supported Browsers:
  • IE(8-11)
  • Chrome(14-39)
  • Firefox(3-34)
  • Safari(4.0-5.1)
  • Opera(10.6-25.0)
  • iOS(3.0-7.0)
  • Android(1.5-4.1)

Chrome Dev Tools

Chrome dev tools, in the latest version of chrome, allows you to emmulate various mobile devices, and change the resolution of the screen itself to allow for testing of the scaling of your webpage. In addition Dev tools allows you to detect memory leaks, and framerates of renderings on your webpage.

Supported Browsers:
  • Chrome

IDEs and Code Editors

Brackets

Brackets is an open-source Web Development IDE. It is mainly a simple text-editor program, but does support the opening of folders, as well as auto-indentation/auto-complete by default. In addition, many 3rd party developers have made extensions for Brackets which extend the functionality of the Program, to make creating HTML/CSS/Javascript documents quicker and easier.

Supported Operating Systems:
  • OS X
  • Windows(32-bit and 64-bit)
  • Ubuntu(32-bit and 64-bit)

Sublime Text 3

Sublime is a text-editor program that is focused towards being used for writing code. Sublime supports a various number of programming languages, and has coloring/auto-indentation/auto-complete functionality for most languages. Sublime3 is currently in beta testing, and is available as a free trial currently.

Supported Operating Systems:
  • OS X
  • Windows(32-bit and 64-bit)
  • Ubuntu(32-bit and 64-bit)

Brackets

A free comprehensive Web Development IDE. The software includes a high quality code assist, deployment wizards, integrated debuggers, git integration, built in terminal, as well as support for customizing the IDE by allowing scripting of custom commands. The program is available as a plugin to Eclipse, or as a stand-alone program, and supports all major web development languages.

Supported Operating Systems:
  • OS X
  • Windows(32-bit and 64-bit)
  • Ubuntu(32-bit and 64-bit)

Recommended Paletts

A color palette is a set of colors that work together to style an interface. The colors should work together nicely to be asthetically pleasing and help the user differentiate between different elements of the page.
paletton.com offers a free color palette creator. It lets you choose between monocrome, adjacent colors(3-colors), triad(3-colors), and tetrad(4-colors)

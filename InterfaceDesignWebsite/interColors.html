Art

Choosing colors for your user interface is a very delicate process. Colors must work together to create an eye-pleasing scheme of colors. When choosing colors for a page it is important to consider the purpose of the page and what emotion you are trying to A good color scheme can invoke emotion within the user and help to divide portions of a page. By the same token, a bad color scheme can be off-putting to the user and can reduce the functionality of your interface.

Recommended Paletts

A color palette is a set of colors that work together to style an interface. The colors should work together nicely to be asthetically pleasing and help the user differentiate between different elements of the page. There are plenty of online resources to help you select a color scheme.
  • paletton.com offers a free color palette creator. It lets you create your own color palette from monocrome, adjacent colors(3-colors), triad(3-colors), and tetrad(4-colors)
  • colourlovers.com lets the user choose from millions of pre-made color combinations created by users.

Cross Platform Issues

The color a user will actually see on their screen can vary based on what type of machine is being used. The default shade of gray differs based on whether the user is using a PC or a UNIX machine. You can make your GUI background color match the default component background color with the following code...

    defaultBackground = get(groot, 'defaultUicontrolBackgroundColor');
    fh.Color = defaultBackground;

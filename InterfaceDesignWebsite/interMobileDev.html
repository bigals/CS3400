Mobile Interfaces must be as scalable and adaptable as possible. Mobile device sizes can vary greatly between tablets and small smartphones.

Manufacturer Guidelines

The major mobile device manufacturers publish guidlines for how their apps should be designed. This helps standardize the interfaces, helping users learn and understand new apps quickly.

  • iOS Human Interface Guidelines
  • Andriod User Interface Guidelines
  • UI Guidelines for BlackBerry 6.0 Smartphones
  • Nokia Developer Design Portal
  • Symbian^3 UI Style Guidelines

User Input

Keyboard
Most mobile devices have several keyboard variations to choose from (text, number, email, URL, etc). Setting this appropriately ensures the user has the most useful keyboard. Other options can be disabled to fit the specific situation like auto-correct, auto-capitalizaiton, and auto-complete.
Gestures
Gestures like swiping and pinching can be used with touch interfaces. These can make it easier for a user to navigate on a small screen.

jQuery Mobile

jQuery Mobile is a touch-optimized web framework designed to be easily compatible with a wide variety of smartphones and tablets. It is built on top of the jQuery core. JQuery handles a lot of the scaling issues that come up when designing interfaces for a wide range of screen sizes.

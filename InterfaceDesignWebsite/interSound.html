Art

An interface designer must be very careful when it comes to adding sound. It is very easy to annoy the user with sounds. Those of us who remember mySpace can attest to the fact that a loud, autoplayed sound on a page is very annoying and distracting. Sounds can be used sparingly when necessary to add character to an interface.

Volume

Volume can be changed in HTML using the DOM volume property. First you must assign the element to a variable using the .getElementById() function. Next you use yourVariable.volume = 0.2. This example would set the volume to 20% (1.0 is the default, max volume).

Cross Platform Compatibility

Not all sound formats are supported by all of the major browsers. This issues can be handled by layering the same audio file in different formats within one <audio> tag. This way, if the browser does not support the first format listed, it will automatically try the next file format listed.

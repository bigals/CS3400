JavaScript Security Issues Links

XSS and CSRF Vulnerabilities
Cross-site Scripting, XSS, is one of the most common JavaScript vulnerabilities, and is the process of manipulating websites to return malicious code that will render on the client-side, in a manner that is determined by the attacker. Cross-site Request Forgery, CSRF, allows attackers to manipulate browsers to take unintended actions on other sites of the attackers choosing.
JavaScript Security Features
The O'Reilly Definitive Guide to JavaScript's 21st chapter is focused on the security feature built into JavaScript, and includes sections that discuss Restricted Features, Same-Origin Policy, and Security Zones/Signed Scripts.
The Browser Security Handbook, on code.google.com
The Browser Security Handbook contains large amounts of information about the security features that can be leverged in JavaScript, in order to create secure code, that is less prone to attack, and safer for the end-user to use.

JavaScript Standardization/Browser Support

ECMAScript is the language that forms the basis of JavaScript. It is standardized by the Ecma International Standards orginization, and the following list of JavaScript versions refrence the revision of the ECMA-262 specification that the version corresponds to.

JavaScript v1.8.5
Javascript version 1.8.5 is the current standard used widely today. The version is based upon ECMAScript 5, and was released in July of 2010. It is supported by Firefox 4 and higher, Internet Explorer 9, Opera 11.60, and Safari 6.00. ECMAScript 5 Specification
JavaScript v1.8.2
JavaScript version 1.8.2 included everything that JavaScript version 1.8.1 had, but various minor updates were also added. The version was released in June of 2009, and was only supported by Firefox 3.6. ECMAScript 3 Specification
JavaScript v1.8.1
JavaScript version 1.8.1 included everything that JavaScript version 1.8 had, but in addition it added support of native JSON, and other various minor updates were also added. The version was supported by Firefox 3.5. ECMAScript 3 Specification
JavaScript v1.8
JavaScript version 1.8 included everything that JavaScript version 1.7 had, but in addition it added support of generator expressions, and expression closures. The version was released in June of 2008, and was supported by Firefox 3.0, as well as Opera 11.50. ECMAScript 3 Specification
JavaScript v1.7
JavaScript version 1.7 included everything that JavaScript version 1.6 had, but in addition it added support of Pythonic Generators, iterators and let. The version was released in October of 2006, and was supported by Firefox 2.0, as well as Google Chrome 28.0.1500.95. ECMAScript 3 Specification
JavaScript v1.6
JavaScript 1.6 was based upon ECMA-262 Edition 3, but added support of array extras, array/string generics, and E4X. The version was released in November of 2005, and the only browser to support JavaScript 1.6 was Firefox 1.5. ECMAScript 3 Specification
JavaScript v1.5
ECMA-262 Edition 3 was the third revision of the ECMAScript standards. It corresponded to JavaScript 1.5, and was released in December of 1999. It was released in November of 2000, and was supported by Firefox 1.0, Internet Explorer 5.5 through 8, Opera 7.0, Safari 3.0-5, and Google Chrome 1.0 through 10.0.666. ECMAScript 3 Specification

JavaScript Debugging Tools

Firebug

Firebug can be used to debug, edit, and monitor a website's HTML, CSS, DOM, XHR, and Javascript. It will log errors, profile function calls, and sort and organize files into a clean list with the 'script file chooser'. It also allows the developer to run their JavaScript line by line, or run an arbitrary block of code.

Chrome Dev Tools

This debugger is built into the Chrome web browser. Features include the ability to insert conditional breakpoints and navaigate through code step by step with execution control. This tool also has the ability to track exceptions and print stack traces. Chrome Dev Tools also offers a pretty print utility which adds line breaks and makes code easier to read and debug.

Firefox JavaScript Debugger

This debugger is a tool built into the Firefox web browser. It allows both local and remote debugging. Another cool feature included in this debugger is the variable popup tool. When the developer hovers over a variable, a window will popup showing the variable's current value.

JavaScript IDEs

Brackets

Brackets is an open-source web development IDE. It has native support for JavaScript, JQuery, CSS, and HTML. Particular features of the IDE include, auto closing of tags, auto tab identation, and element/syntax highlighting. Furthermore, due to Brackets being third party a variety of different plugins can be installed into the IDE to rpovide additional functionality, such as color pickers, and intellisense in JavaScript. By default Brackets will also allow you to live preview your code in the Google Chrome Browser, so that you can see your changes take effect on the fly.

Sublime Text 3

Sublime Text 3 is not an IDE, but rather a text editor, however various open-source plugins can be installed into the text editor to quickly make it feel like a full IDE for JavaScript. Default functionality of the text editor however provides auto-identation, tab-completion, tab lines, and syntax highlighting for not only JavaScript but a variety of languages.

NetBeans

Netbeans has a variety of functionality that it provides for aiding development of JavaScript code. The main functionality points of the IDE for JavaScript include syntax-highlighting, autocompletion, and code folding. In addition Netbeans provides functionality for unit testing via the JsTestDriver. Netbeans also allows for changes made in Chrome dev tools to be saved into the source files of the code, as well as provides a REST viewer. NetBeans integrates source code control with support for Git, Subversion, Mercurial, and CVS. Via the Apache Cordova framework, NetBeans supports easy mobile web development.

Visual Studio 2013

If a windows machine is being used for development, Visual Studio 2013 can be used as an IDE for JavaScript. Defualt functionality that VS provides for JavaScript consists of code-folding, auto-indentation, syntax highlighting, and intellisense. Furthermore, several Visual Studio extensions have been developed to greatly ease the development of not only JavaScript, but also HTML, and CSS. Furthermore, VS2013 support Git integration into the IDE, to provide easy version control of your code solutions.

The Web Essentials extension for VS2013, incorporated with VS2013 upon install, provides a color picker, and other convienent tools to aid in web development.

In addition other extensions, such as the TypeScript extension can be added to VS2013 to allow for the use of the TypeScript Language, a superset of JavaScript that provides easy to use keywords for things like classes, namespaces, and types. Since TypeScript is a superset of JavaScript, it is compiled, and output as clean JavaScript, allowing for easy portability.
